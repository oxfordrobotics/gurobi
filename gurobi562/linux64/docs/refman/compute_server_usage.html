Compute Server Usage

As noted earlier, the Gurobi Compute Server feature is meant to be almost entirely transparent to both the developers and the users of the programs that use it. However, this section points out a few issues that you'll need to be aware of, including setting up a Compute Server client, setting job priorities, performance considerations, callbacks, and a few coding practices for Compute Server. This section also discusses the use of the new distributed parallel algorithms.
