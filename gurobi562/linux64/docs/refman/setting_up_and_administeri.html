Setting Up and Administering a Gurobi Compute Server

Setting up a Compute Server is generally quite straightforward. You have the option of simply following the instructions in the Quick Start Guide for installing a Compute Server license and starting the server. The default settings have been chosen to work well in most usage environment. However, even if you choose to use the defaults, you'll probably want to be aware of the additional options and capabilities described here, including server configuration and remote administration options.
